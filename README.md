# Islamic App (Quran & Sunnah)

A Flutter mobile application that provides easy access to Quran and Sunnah content through a simple and intuitive user interface. The app is designed to help users listen to Quran recitations and Sunnah audio in one organized platform.

## Getting Started

This project is a Flutter-based mobile application focused on clean UI design, audio playback functionality, and structured content navigation. It demonstrates practical mobile app development without backend or API integration.

### Features
- Home screen with Quran and Sunnah sections
- Quran module with multiple reciters selection
- Play Surahs with smooth audio playback
- Sunnah section with audio playback for Hadith content
- Simple, clean, and user-friendly UI
- Offline/local content handling (no API dependency)

#### Project Purpose
- The goal of this project is to provide an accessible Islamic mobile application that allows users to listen to Quran recitations and Sunnah audio easily, with a focus on simplicity, usability, and performance.
